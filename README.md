# GoldenHour Stock Operations Management System
## WIX1002 Fundamentals of Programming Project (25/26)

### 📌 Project Overview
A Java-based management system designed for **GoldenHour**, a retail watch company. This system improves daily operational efficiency by managing stock, sales, and employee attendance.

### ✨ Key Features
* **Authentication**: Secure Login/Logout & Employee Registration.
* **Attendance**: Clock-in/out with automated working hours calculation.
* **Stock Management**: Daily stock counting (Morning/Night) and Stock In/Out tracking.
* **Sales System**: Transaction recording with automated receipt generation (.txt).
* **Search & Edit**: Quick retrieval and modification of stock and sales data.
* **Storage**: Data persistence using CSV/File handling.

### 🚀 Extra Features Included
* **Graphical User Interface (GUI)**: Built with Java Swing/JavaFX for a user-friendly experience.
* **Data Analytics**: Sales performance and stock availability overview.

### 🛠️ Technologies Used
* **Language**: Java 17
* **Concepts**: OOP (Inheritance, Polymorphism), File I/O, Search & Sort Algorithms.
